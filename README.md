A frontend blog application built with Next.js, TypeScript and Tailwind CSS.
Includes blog listing, filtering, pagination, blog creation and SEO-friendly structure.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- React Hook Form
- Zod
- Radix UI

## Features

- Blog listing with filtering and pagination
- Blog creation with form validation and SEO-friendly structure
- Blog SEO metadata
- Blog category filtering
- Blog search with debounce and pagination
- Blog creation form
- Form validation with structured error handling

## Getting Started

To get started, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/MohsenF3/nextjs-blog.git
```

2. Install dependencies:

```bash
pnpm install
```

3. Create a `.env` file in the root directory and add your environment variables:

```bash
# .env
API_URL=http://localhost:3000
```

4. Run the development server:

```bash
pnpm dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.
